CAPÍTULO I: MS-DOS

Consultora Mentores - Comandos y Soluciones de Excelencia

Problema 1

¿Cuál es el comando en DOS para mostrar todos los archivos que tengan de 1 a 4 caracteres de forma paginada?

Solución

dir ????.* /p

Explicación

El comando dir lista el contenido de un directorio. El signo de interrogación ? actúa como comodín para un solo carácter. Al usar ????.*, se buscan archivos con nombres de hasta cuatro caracteres y cualquier extensión. El parámetro /p hace que la salida se muestre página por página.

Problema 2

¿Cuál es el comando DOS para listar solo directorios de la unidad de D: y ordenados alfabéticamente por nombre de carpeta?

Solución

dir D:\ /ad /on

Explicación

El comando dir D:\ especifica que se liste el contenido de la unidad D:. El parámetro /ad filtra la salida para mostrar únicamente los directorios. Finalmente, /on ordena los resultados alfabéticamente por nombre.

Problema 3

¿Qué hace el siguiente comando: attrib +r D:\COM-99\COLEGIO\COMPUTACIÓN\*.php?

Solución

Establece el atributo de solo lectura a todos los archivos con extensión .php que se encuentran en la ruta especificada.

Explicación

El comando attrib modifica los atributos de los archivos. El signo + añade un atributo, y la r corresponde al de "solo lectura" (read-only). Esto protege a los archivos de ser modificados o eliminados.

Problema 4

¿Cuál es el comando para buscar exactamente la cadena “Cesar” en todos los archivos con extensión .txt de una carpeta?

Solución

find "Cesar" *.txt

Explicación

El comando find busca una cadena de texto específica dentro de uno o más archivos. La cadena a buscar se especifica entre comillas. *.txt indica que la búsqueda se debe realizar en todos los archivos que tengan la extensión .txt en el directorio actual.

Problema 5

¿Cuál es el comando para buscar exactamente la cadena “informe” en todos los archivos con extensión .txt de una carpeta?

Solución

find "informe" *.txt

Explicación

De manera similar al caso anterior, el comando find se utiliza para localizar la cadena "informe" dentro de todos los archivos con extensión .txt del directorio current.

Problema 6

Escriba los comandos necesarios para realizar lo siguiente:

  • a) Limpiar la pantalla.
  • b) Mostrar la fecha y hora del sistema.
  • c) Mostrar la versión del sistema operativo.
  • d) Mostrar la etiqueta del volumen de la unidad C:\

Solución

a) cls

b) date y luego time

c) ver

d) vol C:

Explicación

  • a) cls (Clear Screen) borra todo el contenido de la pantalla de la consola.
  • b) date muestra la fecha actual y permite cambiarla, mientras que time hace lo propio con la hora.
  • c) ver (Version) muestra la versión del sistema operativo MS-DOS que se está ejecutando.
  • d) vol (Volume) muestra la etiqueta del volumen y el número de serie del disco especificado.

Problema 7

En DOS escriba el comando para listar ordenadamente por tamaño de forma descendente todos los archivos y directorios cuyos nombres empiecen con la letra d.

Solución

dir d* /o-s

Explicación

dir d* lista todos los archivos y directorios que comienzan con la letra 'd'. El parámetro /o se usa para ordenar, y el sufijo -s indica que el ordenamiento será por tamaño (size) de forma descendente (el guion invierte el orden).

Problema 8

¿Cuál es el comando en DOS para mostrar todos los archivos que tengan de 1 a 4 caracteres de forma paginada?

Solución

dir ????.* /p

Explicación

El comando dir lista el contenido de un directorio. El signo de interrogación ? es un comodín para un solo carácter. ????.* busca archivos con nombres de hasta cuatro caracteres y cualquier extensión. El parámetro /p pausa la visualización cada vez que la pantalla se llena.

Problema 9

¿Cuál es el comando DOS para listar solo directorios de la unidad de D: y ordenados alfabéticamente por nombre de carpeta?

Solución

dir D:\ /ad /on

Explicación

dir D:\ indica la unidad a listar. El parámetro /ad filtra para mostrar solo directorios (Attribute: Directory). El parámetro /on ordena los resultados por nombre (Order by: Name).

Problema 10

¿Qué hace el siguiente comando: more pmd_pdf.php /c?

Solución

Muestra el contenido del archivo pmd_pdf.php página por página, limpiando la pantalla antes de mostrar cada nueva página.

Explicación

El comando more se utiliza para mostrar el contenido de un archivo de forma paginada. El parámetro /c (clear) tiene la particularidad de limpiar la pantalla antes de visualizar cada página del contenido.

Problema 11

Determinar las instrucciones correctas para listar los archivos con nombres de 5 caracteres y de extensión word.

Solución

dir ?????.doc

Explicación

Se utiliza el comando dir. Los cinco signos de interrogación ????? actúan como comodines para cualquier nombre de archivo que tenga exactamente cinco caracteres. .doc era la extensión estándar para documentos de Word en la era de MS-DOS.

Problema 12

Determinar las instrucciones correctas para listar los archivos con nombres de 5 caracteres y que tengan la extensión de dos caracteres.

Solución

dir ?????.??

Explicación

Nuevamente, ????? especifica un nombre de archivo de cinco caracteres. Los dos signos de interrogación en la extensión .?? indican que se buscan archivos cuya extensión tenga exactamente dos caracteres.

Problema 13

¿Cuál es el comando que permite modificar el contenido de un archivo de texto?

Solución

edit nombre_archivo.txt

Explicación

El comando edit abre el editor de texto de MS-DOS a pantalla completa, permitiendo crear o modificar archivos de texto de una manera más amigable que la línea de comandos.

Problema 14

Para ver el contenido de un directorio y el de los directorios internos ¿qué comando se utiliza?

Solución

dir /s

Explicación

El comando dir con el parámetro /s (subdirectories) lista todos los archivos y directorios del directorio actual y de todos sus subdirectorios de forma recursiva.

Problema 15

¿Qué acción realiza la siguiente instrucción type PRUEBA.TXT >>LPT1?

Solución

Envía el contenido del archivo PRUEBA.TXT al puerto de la impresora (LPT1) para ser impreso.

Explicación

El comando type muestra el contenido de un archivo. El operador de redirección >> añade la salida a un archivo o dispositivo. LPT1 es el nombre reservado para el primer puerto de impresora paralelo. En esencia, este comando imprime el archivo.

Problema 16

¿Cuál es el significado de MS-DOS?

Solución

Microsoft Disk Operating System (Sistema Operativo de Disco de Microsoft).

Explicación

MS-DOS fue un sistema operativo de línea de comandos desarrollado por Microsoft, que fue el estándar para las computadoras personales compatibles con IBM PC durante las décadas de 1980 y 1990.

Problema 17

¿Cuál es el comando correcto para copiar todos los archivos con la extensión .txt en un solo archivo llamado LIBRO.txt?

Solución

copy *.txt LIBRO.txt

Explicación

El comando copy se utiliza para copiar archivos. *.txt selecciona todos los archivos con la extensión .txt. Al especificar un nombre de archivo como destino, el comando concatena (une) el contenido de todos los archivos de origen en ese único archivo de destino.

Problema 18

¿Cuál es el comodín para sustituir a un nombre?

Solución

El asterisco *.

Explicación

El asterisco * es un comodín que representa cualquier número de caracteres (de cero a varios) en un nombre de archivo o extensión. Por ejemplo, C*.* se refiere a todos los archivos que empiezan con la letra C.

Problema 19

¿Cuál es el comando correcto para crear un archivo TOTAL.TXT que contiene los contenidos de los archivos SOFT.TXT y HARD.TXT que se encuentran en la raíz de la unidad D:?

Solución

copy D:\SOFT.TXT + D:\HARD.TXT D:\TOTAL.TXT

Explicación

El comando copy puede utilizar el signo + para concatenar (unir) varios archivos en uno solo. En este caso, se indica que se copie el contenido de D:\SOFT.TXT más el contenido de D:\HARD.TXT en un nuevo archivo de destino llamado D:\TOTAL.TXT.

Problema 20

¿Cuál es el comando correcto para crear un archivo TOTAL.TXT que contiene los contenidos de los archivos SOFT.TXT y HARD.TXT que se encuentran en la raíz de la unidad D:?

Solución

copy D:\SOFT.TXT + D:\HARD.TXT D:\TOTAL.TXT

Explicación

Este problema es idéntico al anterior. Se utiliza el comando copy con el signo + para combinar los dos archivos de origen (D:\SOFT.TXT y D:\HARD.TXT) en un único archivo de destino (D:\TOTAL.TXT).

Problema 21

Elabora el siguiente árbol de directorio y resuelve las preguntas.

Inciso a) Mostrar toda la información del directorio PREFAS ordenado por fecha.

Solución

dir PREFAS /od

Explicación

El comando dir PREFAS lista el contenido del directorio especificado. El parámetro /o se usa para ordenar, y la letra d indica que el criterio de ordenación es la fecha y hora, de la más antigua a la más reciente.

Inciso b) Eliminar el directorio MAT99.

Solución

rd MAT99\CARPETA rd MAT99

O usando un comando más potente (no disponible en todas las versiones de MS-DOS):

deltree /y MAT99

Explicación

El comando rd (o rmdir) solo puede eliminar directorios que están completamente vacíos. Dado que MAT99 contiene la carpeta CARPETA, primero se debe eliminar el contenido (rd MAT99\CARPETA) y luego el directorio padre (rd MAT99). El comando deltree es más directo, ya que elimina un directorio y todo su contenido, incluyendo subdirectorios. La opción /y confirma la eliminación sin pedir confirmación.

Inciso c) Crear 3 archivos vacíos con los nombres arch1.txt, arch2.dat, arch3.sh.

Solución

type nul > arch1.txt type nul > arch2.dat type nul > arch3.sh

Explicación

El comando type nul no produce ninguna salida. El operador de redirección > envía esa "nada" a un archivo. Si el archivo no existe, lo crea. Esto resulta en la creación de un archivo de 0 bytes, es decir, un archivo vacío.

Inciso d) Crear un archivo datos.dat con el contenido: dat1, dat2, dat3 y otro archivo datos2.txt con el contenido: r1, r2, r3. Enviar el contenido del archivo datos.dat al final del archivo datos2.txt.

Solución

echo dat1, dat2, dat3 > datos.dat echo r1, r2, r3 > datos2.txt type datos.dat >> datos2.txt

Explicación

  1. echo [texto] > [archivo] crea un nuevo archivo con el texto especificado.
  2. type [archivo_origen] muestra el contenido de un archivo.
  3. El operador de redirección >> (doble mayor que) se utiliza para anexar (agregar al final) la salida de un comando a un archivo existente sin sobreescribir su contenido original.

Problema 22

Cree la siguiente estructura de directorios en la unidad D:\.

Solución

D: md COM99 md COM99\APLIC md COM99\SO md COM99\VARIOS md COM99\APLIC\WORD md COM99\APLIC\ACCESS md COM99\APLIC\EXCEL md COM99\APLIC\WORD\TEXTO md COM99\APLIC\WORD\NOTAS md COM99\APLIC\EXCEL\TABLAS md COM99\APLIC\EXCEL\GRAFICOS md COM99\SO\DOS md COM99\SO\LINUX md COM99\SO\WINDOWS

Explicación

El comando md (o mkdir, Make Directory) se utiliza para crear un nuevo directorio. Se debe especificar la ruta completa donde se desea crear cada nueva carpeta para construir la estructura de árbol requerida.

Problema 23

¿En el MS-DOS para qué sirve date?

Solución

Sirve para mostrar la fecha actual del sistema y, opcionalmente, para establecer una nueva fecha.

Explicación

Al ejecutar el comando date, la consola muestra la fecha actual y solicita al usuario que ingrese una nueva. Si el usuario simplemente presiona la tecla "Enter", la fecha actual se mantiene sin cambios.

Problema 24

¿En el MS-DOS para qué sirve sort?

Solución

Sirve para ordenar líneas de texto de una entrada de datos (usualmente un archivo) y mostrar el resultado ordenado en la pantalla.

Explicación

sort es un comando de filtro que lee datos, los ordena alfabéticamente (o numéricamente) línea por línea, y luego escribe el resultado en la salida estándar. Por ejemplo, sort < miarchivo.txt mostraría las líneas de "miarchivo.txt" ordenadas alfabéticamente.

Problema 25

Crear la siguiente estructura de directorios en la unidad C:\, verificar visualizando en forma gráfica.

Solución

Creación de la estructura:

C: md prefas md prefas\mate md prefas\fisica md prefas\com99 md prefas\com99\dos md prefas\com99\linux md prefas\com99\html

Verificación gráfica:

tree c:\prefas

Explicación

Se utiliza una serie de comandos md para crear cada uno de los directorios. Para la verificación, el comando tree es la herramienta ideal, ya que muestra la estructura de carpetas a partir de una ruta específica en un formato de árbol visualmente fácil de entender.

Inciso a) Crear el archivo de texto empleado.txt dentro del subdirectorio fisica.

Solución

echo. > c:\prefas\fisica\empleado.txt

Explicación

El comando echo. seguido del operador de redirección > es una forma rápida de crear un archivo de texto vacío en la ruta especificada.

Inciso b) Copiar los archivos que termine con la letra “o” en el nombre y la extensión sea ‘txt’, del directorio fisica al directorio com99.

Solución

copy c:\prefas\fisica\*o.txt c:\prefas\com99

Explicación

El comando copy se usa para duplicar archivos. *o.txt es el patrón de búsqueda: el asterisco * es un comodín que representa cualquier cantidad de caracteres, por lo que *o.txt se traduce como "cualquier archivo cuyo nombre termine en 'o' y tenga la extensión .txt". El segundo parámetro es la carpeta de destino.

Inciso c) Establecer los atributos de solo lectura y oculto al archivo empleado.txt del directorio c:\prefas\fisica.

Solución

attrib +r +h c:\prefas\fisica\empleado.txt

Explicación

El comando attrib gestiona los atributos de los archivos. El signo + se usa para añadir un atributo. +r activa el atributo de solo lectura (Read-only) y +h activa el atributo de oculto (Hidden).

Inciso d) Buscar a los empleados que son de “tarija” del archivo empleado.txt.

Solución

find "tarija" c:\prefas\fisica\empleado.txt

Explicación

El comando find busca una cadena de texto exacta (sensible a mayúsculas y minúsculas) dentro de un archivo. El primer parámetro es la cadena a buscar entre comillas, y el segundo es el archivo donde se realizará la búsqueda.

Problema 26

¿Cuál es el comando que permite establecer los permisos de solo lectura y archivo oculto a datos.txt?

Solución

attrib +r +h datos.txt

Explicación

Al igual que en el ejercicio anterior, se utiliza el comando attrib. +r añade el atributo de solo lectura para prevenir su modificación, y +h añade el atributo de oculto para que no aparezca en listados de directorio normales (a menos que se use dir /a).
